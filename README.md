# 🌾 AgriSahayak 3.0 – Smart Farmer Assistance & Government Scheme Intelligence Platform

**Team:** SmartStruct  
**Problem Statement:** Simplify farmer access to schemes, insurance, and financial support using minimal inputs.

AgriSahayak is a smart web-based platform designed to help farmers easily discover, understand, and apply for government schemes, insurance, and financial support. The system reduces complexity, removes middlemen dependency, and provides a structured, digital, and transparent workflow for both farmers and administrators.

---

## 🚀 Features

### 👨‍🌾 Farmer Panel
- 🤖 Smart chat assistant for guidance
- 🌾 Crop & damage-based scheme suggestions
- 🧾 Minimal input flow (crop, issue, land, documents)
- 📂 Secure document upload (Aadhaar, PAN, Land Records, etc.)
- 👤 Profile management with synced document status
- 📊 Personalized dashboard
- 📝 Apply for schemes with controlled flow
- 🌍 Multi-language support (English / Hindi / Marathi)
- 💳 Subscription system:
  - Free Plan: Limited chats per day  
  - Premium Plan: Unlimited chat + priority assistance  

### 🧑‍💼 Admin Panel
- 📋 View all farmer applications
- ✅ Approve / ❌ Reject requests
- 🕒 Track pending and processed applications
- 📊 Monitor scheme usage and requests
- 🔔 Real-time updates

---

## 🧠 Smart Chat Flow

1. Detect Crop  
2. Detect Damage  
3. Ask Land Area  
4. Ask Document Availability  
5. Suggest Best Scheme  
6. Show **Apply** button (only when all conditions are met)
